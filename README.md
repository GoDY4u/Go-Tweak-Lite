## üì¶ Installation & Usage

### Quick Installation

Run this command in **PowerShell (Administrator)**:

```powershell
irm https://raw.githubusercontent.com/GoDY4u/Go-Tweak-Lite/main/install.ps1 | iex
```

### Manual Installation

1.  Download the latest release
    
2.  Extract to your preferred directory
    
3.  Run **PowerShell as Administrator**
    
4.  Execute: `.\Go-Tweak.ps1`
    

### Command Line Options

```powershell
.\Go-Tweak.ps1 -Quiet          # Silent mode with logging
.\Go-Tweak.ps1 -LogPath "C:\path\to\logs.log"  # Custom log location
```
